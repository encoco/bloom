## 📝 소개
Bloom은 현대 소비자들의 온라인 쇼핑 니즈를 충족시키기 위한 의류 쇼핑몰 웹사이트입니다. 사용자 중심의 UI/UX를 통해 편리한 쇼핑 경험을 제공하며, 다양한 의류 상품을 효율적으로 탐색하고 구매할 수 있습니다.
